Web hosting

El web hosting es un servicio que les proporciona a los usuarios de internet un sistema para alojar contenido de cualquier tipo (información, imágenes, video) accesible vía web.

En mi caso, he utilizado el servicio DriveToWeb.

Lo que hice fue, primero ajustar la carpeta "web" para que fuera pública.

Después descargué la carpeta "web", y la subí a Google Drive.

Por último, fui a la página de DriveToWeb, y generé los enlaces a la página web.
